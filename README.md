# ⚙반응형 웹 UI 프로젝트

HTML, CSS, JavaScript, jQuery를 활용한 반응형 웹 UI 프로젝트입니다.

사용자에게 직관적인 네비게이션, 동적 콘텐츠 로딩, 슬라이드 쇼 및 비디오 플레이어 기능을 제공합니다.


## 📌주요기능
- 반응형 메뉴 (모바일/데스크탑 구분)
- 메인 슬라이더 (Owl Carousel 적용)
- 상품 슬라이드 (브레이크포인트 설정)
- 스크롤 트리거 애니메이션 (GSAP 사용)
- 커스텀 마우스 커서 및 호버 효과
- Top 버튼 기능
- 디바이스별 이미지 자동 교체
